# qrmaster
qrmaster - QR and Bar code detect, open, create for pc

# Структура:
## tkinker app для создания и распознования кодов
### Создание:
#### Библиотека qrcode и вывод на экран в tkinker app
### Чтение:
#### Цикл приема кадров с камеры и cv2.QRCodeDetector(), cv2.barcode.BarcodeDetector() для распознования
#### Открытие с помощью webbrowser
